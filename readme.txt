The Verlico plugin automatically adds real-time content analytics to your blog. See who’s is actually reading your posts and how long they stay.


== Description ==
[Verlico](http://verlico.com) is a content analytic platform for your blog. Once you [become a Verlico member](http://verlico.com/request), you can use this plugin to automatically add the necessary JavaScript to your WordPress blog. After installing the plugin, you’ll instantly see your site’s traffic and audience behavior.

== Installation ==
1. Upload `verlico.php` to the `/wp-content/plugins/` directory
2. Activate the plugin through the \'Plugins\' menu in WordPress
3. Under Settings > Verlico, add your Verlico Username

== Frequently Asked Questions ==
= What is Verlico? =

We make software to help bloggers understand their audiences and craft relevant content.

Use Verlico (and this plugin) to see how many people read your blog posts and what content has the most value to your readers. Check it out for yourself and sign up for [free](https://verlico.com/request/)!

= What does this plugin do? =

It’s an easy way for you to install the Verlico code you need in order to see what\'s happening on your WordPress site. Make sure you [sign up for Verlico](http://verlico.com/request) and have an account first, or this plugin won’t work for your site.

= Will this plugin slow down my site? =

Nope. Verlico code is completely asynchronous, meaning it doesn\'t begin to run until everything else on your page has already loaded.
